# Qiu Nuxt Starter

我的 Nuxt 初始模板。

[点我跳转 Nuxt 文档](https://nuxt.com/docs/getting-started/introduction)。

## 1. 项目启动

### 1.1 安装依赖

```bash
pnpm install
```

## 1.2 启动开发服务器

```bash
pnpm dev
```

## 1.3 构建生产制品

```bash
pnpm build
```

本地预览生产制品：

```bash
pnpm preview
```

[点我跳转部署文档](https://nuxt.com/docs/getting-started/deployment)。
